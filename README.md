# Unemployment Analysis with Python (COVID-19 Impact)

## 📌 Project Overview
This project analyzes unemployment trends in India using Python. The objective is to understand how unemployment rates changed over time, especially during the COVID-19 pandemic, through data cleaning, visualization, and exploratory data analysis (EDA).

## 🎯 Objectives
- Clean and preprocess unemployment data.
- Analyze unemployment trends across different states.
- Study the impact of COVID-19 on unemployment rates.
- Visualize unemployment patterns using charts.
- Generate meaningful insights from the data.

## 🛠️ Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Jupyter Notebook

## 📂 Dataset
The dataset contains unemployment statistics across different states in India, including:
- State
- Date
- Estimated Unemployment Rate (%)
- Estimated Employed
- Estimated Labour Participation Rate (%)
- Region

## 📊 Project Workflow
1. Import required libraries.
2. Load the dataset.
3. Perform data cleaning.
4. Conduct Exploratory Data Analysis (EDA).
5. Create visualizations.
6. Analyze the impact of COVID-19.
7. Summarize key findings.

## 📈 Visualizations
The project includes:
- Unemployment Rate Trend
- State-wise Unemployment Comparison
- Regional Analysis
- Distribution of Unemployment Rate
- COVID-19 Impact Analysis

## 🔍 Key Insights
- Unemployment rates increased significantly during the COVID-19 lockdown period.
- Some states were more affected than others.
- Regional differences in unemployment trends were observed.
- Data visualization helps identify economic patterns effectively.

## ▶️ How to Run
1. Clone the repository.
2. Install the required libraries:
   ```bash
   pip install pandas numpy matplotlib seaborn
   ```
3. Open the Jupyter Notebook:
   ```bash
   jupyter notebook
   ```
4. Run all cells.
